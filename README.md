## Technologies

- React
- CSS
- TypeScript

## Getting Started

1.  Clone: `git clone https://github.com/arjunvijay03/landing-page.git`
2.  Install: `npm install` or `yarn install`
3.  Run: `npm run dev` or `yarn dev`
